🚀 AI Resume Analyzer 🤖

AI-powered web app to analyze resumes, match skills with job roles, and provide intelligent career guidance.

📌 Overview

AI Resume Analyzer is a smart web application that helps users evaluate their resumes against a target job role.

It provides:

Skill match analysis
Missing skill recommendations
AI-based career guidance
Downloadable PDF reports
Secure OTP-based login system
